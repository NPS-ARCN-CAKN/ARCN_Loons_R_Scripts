# Data processing and reporting R scripts for the National Park Service, Yellow-billed loons monitoring program

This repository contains scripts in R, R Markdown and Quarto for processing loon survey data for quality and summarization.

[Yellow-billed loons monitoring program](https://www.nps.gov/im/arcn/ybl.htm)

[Reports and analytical products](https://irma.nps.gov/DataStore/Reference/Profile/2216938)
